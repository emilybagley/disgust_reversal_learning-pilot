Pilot data and analysis
This folder contains the pilot data and the analysis of that data - creating results referenced in the stage 1 Registered Report

List of files:
1_create_clean_dataframes: A datacleaning file which applies exclusion criteria --> finds that 3/14 participants would have been excluded due to inattention
2_video_ratings_analysis: tests whether individualised stimulus selection procedure has worked - i.e., whether stimuli are matched in valence and arousal

These files call functions in task_func.py and vid_ratings_func.py files
